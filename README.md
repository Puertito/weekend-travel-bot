# Weekend Travel Bot

Telegram-бот для поиска дешевых авиабилетов на выходные из Tenerife Norte (TFN).

## Возможности

- Поиск билетов TFN → BCN
- Поиск билетов TFN → MAD
- Вылет в пятницу
- Возврат в воскресенье
- Уведомления в Telegram при цене ≤ заданного лимита
- Автоматический запуск через GitHub Actions

## План развития

- История изменения цен
- Несколько источников поиска
- Поддержка новых направлений
- Аналитика лучших цен
